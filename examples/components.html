Components styling

1. Default Buttons

Button styles can be applied to anything with the .btn class applied. However, typically you'll want to apply these to only <a> and <button> elements for the best rendering.

Button sizes

Fancy larger or smaller buttons? Add .btn-large, .btn-small, or .btn-mini for additional sizes.

Button styles

There are two types of buttons styles included in Kickoff, 'Stylish' & 'Flat'. There are a few global .scss vars that will determine the style for the buttons, these are:

  • // 'flat' or 'stylish'
    $kickoff-button-style : 'stylish';
  • $kickoff-button-border-radius : 5px;
  • $kickoff-button-border-width : 0px;

    .SCSS 
    .btn {
	@include kickoff-button;
}
.btn-primary {
	@include kickoff-button(#fff, $blue);
}
.btn-warning {
	@include kickoff-button(#fff, lighten(#444, 10%));
}
.btn-flat {
	// Button vars can be set on individual buttons
	$kickoff-button-style : 'flat';
	@include kickoff-button(#fff, $blue);
}
